# M1 交付报告 —— 荒岛与别墅场景

《无人生还：士兵岛》第一人称 3D 叙事推理游戏，里程碑 M1。
纯程序化几何体（Three.js r160，本地 vendored），低多边形 + 铅灰/墨绿/暗红木/烛光橙色板，Art Deco 风格。

## 2. 运行方式

- 双击 `启动游戏.bat`（自动起服并打开浏览器）；或
- 在 `game-正式版/` 下 `python -m http.server 8000`，浏览器访问 `http://localhost:8000/`
- URL 参数：
  - `?chapter=N`（0–11）：启动即切到第 N 章天气预设
  - `&play=1`：跳过覆盖层（无 pointer lock，供截图/自动测试）
  - `&pos=x,z&yaw=弧度&y=高度&pitch=弧度`：调试机位
- 控制台 API：
  - `setChapter(N)` / `window.WeatherAPI = { setChapter(n), getChapter(), presets }`
  - `window.PerfAPI = { info: () => renderer.info }`
  - `window.DebugAPI = { teleport(x,z,yaw,y), getState(), move(dx,dz) }`（自动验收用）
- 直接双击 index.html（file://）时 fetch 被浏览器拦截，页面会显示：
  “请用 启动游戏.bat 或 python -m http.server 运行”。

## 3. 场景清单

### 岛屿（半径 100m，可行走边界 95m）

- 解析式高度场：中央平地（别墅区 ~1.5m），西北悬崖台地（+9.2m，台顶 ~10m），
  台地外缘西北扇区陡削成悬崖（坡度 >1.15，下方海面礁石）。
- 悬崖小径：别墅后方 → 台地的之字形折线，高度场沿折线切割/填出平滑坡道，
  路面土色 + 石板踏步（instanced），只有它能上台地。
- 无限海面：900m 大平面 + 顶点波浪着色器（onBeforeCompile 注入，flat shading 风格化浪），
  浪高/海色随天气预设联动；岸边 ~190 块 instanced 礁石 + 70 块白色浪尖。
- 6 个室外点位（均有木柱立牌，canvas 贴图，文案来自 data/places.json）：

| 2 | 北岬角 cape_north | 岛北端 (0, -76) | 礁石岬 + 望海长椅（面向大海） |

### 别墅（24m×16m，三层，层高 3.2m，坐北朝南）

- 外观：方正体块、平屋顶 + 阶梯式女儿墙檐口、层间水平装饰线、圆形舷窗（铜环+玻璃）、
  阶梯式正门雨棚、“士兵岛别墅”名牌（places.json）。
- 一层（地面 1.8m）：大厅（壁炉+大理石壁炉台【10 个瓷人空位底座，instanced】、
  留声机【黄铜大喇叭】、大理石雕像、圆形餐桌+11 把椅子、吊灯、扶手椅）、
  餐厅（长桌+6 椅+餐边柜）、厨房（灶台/橱柜）、管家房、后廊、露台（Deco 柱式矮栏）。
- 二层（5.0m）：走廊 + 十间客房（每间：床/书桌/椅/床头柜，instanced；
  门口 3D 小牌 + canvas 名牌，文字全部来自 places.json 的 bedrooms）+ 公共浴室（浅蓝瓷砖、
  浴缸/洗手盆/镜子）；维拉房间窗台放白色大理石熊钟（球体拼熊形 + 钟面指针）。
- 三层（8.2m）：书房（满墙书架 15 组 + 480 本 instanced 书脊【暗红/棕/墨绿/灰】、
  扶手椅×2、大书桌、暗红褶皱窗帘×4）+ 储藏室（木箱×12、罐头×24 instanced）。
- 楼梯：西墙楼梯间双跑折返（16 级×2 跑，步高 0.2m 自动上台阶），三层贯通；
  实心阶梯体、中间栏墙、平台护栏；梯下净高 ≥2.85m。
- 所有门洞均为开启状态（无门板），每个房间都可走进。

## 4. 控制与碰撞方案

- 第一人称：WASD 移动、Shift 奔跑、鼠标环顾（俯仰 ±85°）、重力 20m/s²、
  步高 0.35m（上下台阶自动吸附，楼梯可走）。
- 碰撞（collision.js）：
  - 静态 AABB 列表（墙体/家具/栏杆/烟囱等 ~300 盒），玩家胶囊体（r=0.35，h=1.7）
    在 XZ 平面做圆形推出（两遍迭代），天然沿墙滑动；
  - 可行走平台列表（楼板/台阶/码头板），地面高度 = max(地形, 可踏上平台顶)；
  - 解析地形高度场 + 数值梯度：坡度 >1.15 的方向移动被拦截（悬崖“隐形护栏”，
    只有小径坡道能上台地，悬崖边摔不下去）；地形高度 <0.25 为海水，不可进入；
  - 岛屿边界圆柱 r=95m（码头栈道等平台上豁免，可走到伸入海中的栈道尽头）。
- 楼梯碰撞设计：台阶只做“平台”不做实心盒（避免上一级盒体挡路），
  配合 0.35m 步高逐级上升；梯底/梯侧由封挡盒与栏墙防碰头、防跌落。

## 6. 性能实测（renderer.info，CDP 无头 Chrome 抓取）

| 机位/预设 | 三角面 | draw calls |
|-----------|--------|------------|
| 码头出生点全景（ch 0） | 83,578 | 103 |
| 暴雨（ch 2） | 82,894 | 98 |
| 大厅室内（ch 6） | 80,122 | 50 |
| 悬崖小径视角（ch 8） | 48,568 | 10 |
| 海滩视角（ch 9） | 79,960 | 30 |

- 预算：三角面 < 30 万（上限 80 万）、draw calls ~150 内 → **全部达标**（峰值 8.4 万面 / 103 calls）。
- 手段：墙体/地板/装饰按材质合并（每批 1 次 draw call）；重复物件全部 InstancedMesh
  （礁石/浪尖/石板/灌木/圆木/书脊/瓷人底座/床/椅/桌/柜/木箱/罐头）；
  无后处理（fog + ACES tone mapping + CSS 暗角）；点光仅 7 盏且无阴影，阳光单张 2048 阴影图。
- 控制台 `PerfAPI.info()` 可随时复查。

## 7. 数据驱动（文案 JSON 化）

- `data/places.json`：别墅名、10 间客房名牌+英文副标、8 个公共房间名牌、6 个室外点位名牌
  → 3D 门口小牌（canvas serif 贴图）、室外立牌、HUD 点位提示全部读取，JS 中无硬编码。
- `data/chapters.json`：章节标题/氛围/日期 → HUD 章节卡。
- `data/ui.json`：controls（移动/环顾/交互/暂停）→ 覆盖层与右下角操作提示；
  gameTitle/warning/loading → 覆盖层文案。
- fetch 失败（file:// 直开）→ 页面显示明确中文提示。

## 11. 遗留问题

- 无 NPC、无剧情系统（M1 范围外）；E 交互仅点位提示（note/sub 弹出 toast）。
- 客房 interior 仅基础家具占位（风格统一的 Deco 暗红木），M2 可做差异化陈设。
- 无头软渲染下帧率不代表真机；真机中端独显/核显预算内（8.4 万面 / ~100 calls）。
- 海面为单色材质 + 顶点浪，无天光反射（M1 风格化可接受）。
