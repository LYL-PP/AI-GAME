# 无人生还：士兵岛（And Then There Were None: Soldier Island）

第一人称 3D 叙事推理游戏 Demo。改编自阿加莎·克里斯蒂《无人生还》。
Three.js r160 · 纯程序化几何体 · Web Audio 程序合成 · 无外部模型/音频文件。

## 运行方式

- 双击 `启动游戏.bat`（自动起服并打开浏览器），或
- 在本目录执行 `python -m http.server 8000`，浏览器访问 `http://localhost:8000/`
- 直接双击 `index.html`（file://）无法加载数据，会提示改用上述方式
- 调试参数：`?chapter=N`（0–11 直达章节）、`&play=1`（跳过覆盖层）

## 流程与结局条件

- 序章：码头 → 门廊触发晚餐 → 留声机指控 → 自由探索。
- 第 1–10 章：每章一起死亡事件。跟随 HUD 箭头前往现场，E 检视尸体与关键物品（13 条线索）。
- 推理板（Tab）：拖线索到嫌疑人卡上连线自组织；第 3 章起可"标记怀疑"（软指认，会改变对应 NPC 对话）。
- 终章（第 11 章）：推理板出现终局指认槽——拖 1 名嫌疑人 + 最多 3 条线索。
  - **真结局 · 瓶中信**：指认 **沃格雷夫**，且证据含 clue_08 / clue_09 / clue_10 中 **任意两条**。
  - **结局 B · 错判**：指认他人，或证据不足（系统会指出时间线矛盾）。
  - **结局 C · 沉默**：10 分钟未提交，或点"放弃指认"。
  - **隐藏结局 · 第十一人**：真结局 + 集齐 13/13 线索 + 全程零软指认误判（软指认过任何非沃格雷夫的人即失效）。

## 系统特色

- **十声部递减配乐**：Web Audio 程序合成，每章结算后按 audio.json 顺序永久撤掉一件乐器（HUD 右上角声部计），第 10 章后单乐器残响，终章全静默。结算仪式：音乐全停 → 瓷人碎裂 → 3 秒静默 → 新编制淡入。
- **空掉的岛**：每死一人，大厅圆桌撤一把椅子一副餐盘；第 6 章后布伦特的毛线从摇椅消失；NPC 逐个从日程中移除。
- **天气演出**：12 套天气/灯光预设（阴天→风暴→浓雾→烛光夜→过曝烈日→金色黄昏），海面浪高联动。
- **二周目批注**：通关一次后，统计页可开启"凶手视角批注"——重玩时每章结算卡底部浮现小字。

## 已知限制

- 立绘为占位 SVG（正式立绘提示词见 `docs/立绘与参考图提示词.md`）。
- 音频为程序合成（音色近似，非实录）；留声机 TTS 依赖浏览器 speechSynthesis，失败时退回蜂鸣。
- 单存档槽（localStorage `attwn_save`），无存档管理界面。
- NPC 寻路为区域图直线航点，无相互避让；死亡现场为定格+剪影，无血腥建模。
- 海面为风格化顶点浪，无反射。
